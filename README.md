# Blissful

A **modern, responsive e-commerce website** offering a curated selection of **beauty and skincare products**, designed for a luxurious and effortless shopping experience. With a sleek, user-friendly interface, it ensures seamless navigation and secure checkout.

## Key Features
- **Responsive Design:** Works flawlessly on desktops, tablets, and mobile devices, providing a smooth shopping experience.  
- **Product Catalog:** Browse, search, and filter a curated selection of beauty and skincare products.  
- **Shopping Cart & Checkout:** Secure checkout process with add-to-cart functionality and order summary.  
- **Admin Dashboard:** Manage products, categories, and orders efficiently.  
- **Payment Integration:** Supports secure payment gateways for hassle-free transactions.  
- **Deployment Ready:** Hosted on Vercel for live demo and fast performance.  

## Technologies Used
React.js | Node.js | Express.js | MongoDB | Tailwind CSS / Bootstrap
